# Consistency Rubric

## Overview
This rubric evaluates the uniformity and coherence of style, terminology, formatting, and presentation throughout educational materials, ensuring a professional and cohesive learning experience.

## Scoring Scale
- **4 - Exemplary**: Perfect consistency creating seamless experience
- **3 - Proficient**: Good consistency with minor variations
- **2 - Developing**: Noticeable inconsistencies affecting quality
- **1 - Inadequate**: Major inconsistencies causing confusion

## Evaluation Criteria

### 1. Terminology and Language (25%)

#### Exemplary (4)
- Perfect terminology consistency throughout
- Unified voice and tone across all content
- Consistent technical vocabulary usage
- Flawless glossary alignment

#### Proficient (3)
- Generally consistent terminology
- Mostly unified voice and tone
- Good technical term usage
- Strong glossary alignment

#### Developing (2)
- Some terminology variations
- Occasional voice/tone shifts
- Inconsistent technical terms
- Partial glossary alignment

#### Inadequate (1)
- Frequent terminology changes
- Inconsistent voice and tone
- Confusing technical term usage
- Poor or no glossary alignment

### 2. Visual Design Consistency (25%)

#### Exemplary (4)
- Perfect visual style unity
- Consistent color schemes throughout
- Uniform iconography and graphics
- Flawless layout patterns

#### Proficient (3)
- Strong visual cohesion
- Generally consistent colors
- Mostly uniform graphics
- Good layout consistency

#### Developing (2)
- Some visual inconsistencies
- Variable color usage
- Mixed graphic styles
- Inconsistent layouts

#### Inadequate (1)
- No visual unity
- Random color schemes
- Clashing graphic styles
- Chaotic layouts

### 3. Formatting Standards (25%)

#### Exemplary (4)
- Perfect formatting uniformity
- Consistent heading styles
- Uniform list and table formatting
- Flawless spacing and alignment

#### Proficient (3)
- Strong formatting consistency
- Generally uniform headings
- Good list/table consistency
- Mostly consistent spacing

#### Developing (2)
- Noticeable formatting variations
- Inconsistent heading usage
- Variable list/table styles
- Irregular spacing patterns

#### Inadequate (1)
- Random formatting throughout
- Chaotic heading structure
- Inconsistent list/table formats
- Poor spacing and alignment

### 4. Instructional Approach (25%)

#### Exemplary (4)
- Perfectly consistent pedagogical style
- Uniform assessment approaches
- Consistent feedback mechanisms
- Seamless learning experience flow

#### Proficient (3)
- Generally consistent pedagogy
- Mostly uniform assessments
- Good feedback consistency
- Smooth learning flow

#### Developing (2)
- Variable pedagogical approaches
- Inconsistent assessment styles
- Mixed feedback quality
- Uneven learning experience

#### Inadequate (1)
- Conflicting pedagogical styles
- Random assessment approaches
- Inconsistent or missing feedback
- Disjointed learning experience

## Implementation Guidelines

### Assessment Process
1. Create consistency checklist from style guide
2. Review samples from throughout content
3. Document all variations found
4. Assess impact of inconsistencies
5. Calculate weighted score

### Common Issues to Address
- **Term creep**: Same concept called different names
- **Style drift**: Gradual departure from standards
- **Template violations**: Not following established patterns
- **Voice shifts**: Changing from formal to informal

### Improvement Recommendations
- Develop comprehensive style guide
- Use templates and style sheets
- Implement regular consistency audits
- Establish review checkpoints
