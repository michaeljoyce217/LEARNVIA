# Mathematical Formatting Rubric

## Overview
This rubric evaluates the presentation quality of mathematical content, including equations, symbols, notation, and visual representation of mathematical concepts.

## Scoring Scale
- **4 - Exemplary**: Perfect mathematical presentation
- **3 - Proficient**: Good mathematical formatting with minor issues
- **2 - Developing**: Adequate formatting but with notable problems
- **1 - Inadequate**: Poor mathematical presentation impeding understanding

## Evaluation Criteria

### 1. Equation Formatting and Layout (25%)

#### Exemplary (4)
- All equations perfectly formatted and aligned
- Consistent use of display vs. inline equations
- Optimal spacing and line breaks
- Clear equation numbering and referencing

#### Proficient (3)
- Most equations well-formatted
- Generally good display choices
- Adequate spacing and breaks
- Good numbering system

#### Developing (2)
- Some formatting inconsistencies
- Mixed quality in display choices
- Occasional spacing issues
- Basic numbering present

#### Inadequate (1)
- Poor equation formatting
- Inappropriate display choices
- Frequent spacing problems
- Missing or confused numbering

### 2. Symbol Usage and Notation (25%)

#### Exemplary (4)
- Perfect symbol consistency throughout
- All notation properly defined
- Standard mathematical conventions followed
- Clear distinction between variables and constants

#### Proficient (3)
- Generally consistent symbols
- Most notation defined
- Usually follows conventions
- Good variable/constant distinction

#### Developing (2)
- Some symbol inconsistencies
- Some undefined notation
- Occasional convention violations
- Sometimes unclear distinctions

#### Inadequate (1)
- Inconsistent symbol usage
- Much undefined notation
- Frequent convention violations
- Confusing variable usage

### 3. Mathematical Typography (25%)

#### Exemplary (4)
- Perfect use of italic/roman fonts
- Correct bold for vectors/matrices
- Proper subscript/superscript formatting
- Excellent Greek letter rendering

#### Proficient (3)
- Good font usage overall
- Usually correct bold usage
- Generally good sub/superscripts
- Good Greek letter display

#### Developing (2)
- Some font inconsistencies
- Occasional bold errors
- Some sub/superscript issues
- Basic Greek letter usage

#### Inadequate (1)
- Poor font choices
- Incorrect or missing bold
- Frequent sub/superscript problems
- Poor Greek letter rendering

### 4. Visual Mathematical Elements (25%)

#### Exemplary (4)
- Perfect graphs and diagrams
- Clear geometric representations
- Excellent matrix/table formatting
- Optimal use of mathematical diagrams

#### Proficient (3)
- Good quality visuals
- Clear geometric figures
- Good matrix/table display
- Appropriate diagram usage

#### Developing (2)
- Adequate visual quality
- Basic geometric representation
- Some matrix/table issues
- Limited diagram usage

#### Inadequate (1)
- Poor quality visuals
- Unclear geometric figures
- Poor matrix/table formatting
- Missing or confusing diagrams

## Implementation Guidelines

### Assessment Process
1. Review all mathematical expressions for formatting
2. Check symbol consistency across document
3. Verify typography standards
4. Evaluate visual mathematical elements
5. Calculate weighted score

### Common Issues to Address
- **Mixed notation systems**: Switching between conventions
- **Font confusion**: Using wrong fonts for mathematical elements
- **Alignment problems**: Misaligned multi-line equations
- **Symbol ambiguity**: Same symbol for different meanings

### Improvement Recommendations
- Use LaTeX or proper equation editors
- Create symbol definition tables
- Follow established style guides (AMS, ISO)
- Implement consistency checking

## Mathematical Formatting Standards

### Equation Types and Usage
- [ ] Display equations for important results
- [ ] Inline equations for simple expressions
- [ ] Aligned equations for derivations
- [ ] Cases notation for piecewise functions
- [ ] Matrix notation properly formatted

### Symbol Conventions
- [ ] Variables in italic
- [ ] Functions in roman
- [ ] Vectors in bold
- [ ] Matrices in bold capitals
- [ ] Constants in roman

### Special Notations
- [ ] Derivatives (prime vs. d/dx)
- [ ] Integrals (limits placement)
- [ ] Summations (index clarity)
- [ ] Set notation (proper symbols)
- [ ] Logic symbols (consistent usage)

### Visual Elements
- [ ] Coordinate axes labeled
- [ ] Units specified
- [ ] Legends provided
- [ ] Scale indicated
- [ ] Colors meaningful

| Element | Font Style | Example |
|---------|------------|---------|

| Numbers | Roman | 1, 2, 3 |
| Functions | Roman | sin, cos, log |
| Vectors | Bold | **v**, **F** |
| Matrices | Bold Capital | **A**, **M** |
| Constants | Roman | e, π, i |
