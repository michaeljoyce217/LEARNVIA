# Mechanical Compliance Rubric

## Overview
This rubric evaluates adherence to established writing conventions, formatting standards, and technical specifications required for educational materials.

## Scoring Scale
- **4 - Exemplary**: Flawless mechanical execution
- **3 - Proficient**: Minor mechanical issues that don't impede understanding
- **2 - Developing**: Noticeable mechanical issues affecting quality
- **1 - Inadequate**: Serious mechanical problems impeding comprehension

## Evaluation Criteria

### 1. Spelling and Typographical Accuracy (25%)

#### Exemplary (4)
- Zero spelling errors throughout document
- No typographical errors
- Consistent use of specialized terminology
- Perfect accuracy in technical terms and names

#### Proficient (3)
- Rare spelling errors (less than 1 per 1000 words)
- Minimal typographical errors
- Generally consistent terminology
- Mostly accurate technical terms

#### Developing (2)
- Occasional spelling errors (2-5 per 1000 words)
- Some typographical errors present
- Inconsistent terminology usage
- Some technical term errors

#### Inadequate (1)
- Frequent spelling errors (more than 5 per 1000 words)
- Many typographical errors
- Inconsistent or incorrect terminology
- Regular technical term mistakes

### 2. Capitalization and Formatting (25%)

#### Exemplary (4)
- Perfect capitalization throughout
- Flawless heading hierarchy
- Consistent formatting styles
- Proper noun usage always correct

#### Proficient (3)
- Generally correct capitalization
- Good heading structure
- Mostly consistent formatting
- Proper nouns usually correct

#### Developing (2)
- Some capitalization errors
- Inconsistent heading usage
- Variable formatting styles
- Occasional proper noun errors

#### Inadequate (1)
- Frequent capitalization errors
- Poor or missing heading structure
- Inconsistent formatting throughout
- Regular proper noun mistakes

### 3. Numerical and Data Presentation (25%)

#### Exemplary (4)
- All numbers formatted consistently
- Perfect table and figure formatting
- Accurate data representation
- Clear and consistent units of measurement

#### Proficient (3)
- Most numbers properly formatted
- Good table and figure presentation
- Generally accurate data
- Usually clear units

#### Developing (2)
- Inconsistent number formatting
- Some table/figure issues
- Occasional data errors
- Sometimes unclear units

#### Inadequate (1)
- Poor number formatting
- Problematic tables/figures
- Frequent data errors
- Confusing or missing units

### 4. Citation and Reference Formatting (25%)

#### Exemplary (4)
- Perfect citation format consistency
- All references complete and accurate
- Flawless bibliography formatting
- Proper attribution throughout

#### Proficient (3)
- Generally consistent citations
- Most references complete
- Good bibliography formatting
- Usually proper attribution

#### Developing (2)
- Some citation inconsistencies
- Some incomplete references
- Basic bibliography present
- Occasional attribution issues

#### Inadequate (1)
- Inconsistent or missing citations
- Many incomplete references
- Poor or missing bibliography
- Frequent attribution problems

## Implementation Guidelines

### Assessment Process
1. Automated spell-check as first pass
2. Manual review for context-specific terms
3. Format consistency check
4. Citation and reference audit
5. Calculate weighted score

### Common Issues to Address
- **Autocorrect errors**: Wrong word substitutions
- **Inconsistent styling**: Mixed formatting within document
- **Number formatting**: Inconsistent decimal places, units
- **Citation gaps**: Missing page numbers, dates, or authors

### Improvement Recommendations
- Use style guides and templates
- Implement peer review process
- Create custom dictionaries for technical terms
- Use reference management software
